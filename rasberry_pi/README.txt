ラズパイメモ
    RPI.GIPO
    output->1  GPIO.HIGH   True
    ラズパイライト
    赤のみ→起動してない
    赤と緑→起動してる(何かファイルに書き込んでる)


自動化
    前進
    壁検知(壁or上り階段)
        上にものがあるか
        ある(壁)
            左右のセンサーの距離の違いで自分と壁の立ち位置を確認
            回転
        ない(上り階段)
            左右のセンサーの距離の違いで階段に対して垂直に
            登る
    下に床がない(左右でどちらかの距離が伸びる)(下り階段)
        左右どちらが距離がのびたか確認
            ↑の結果から垂直になる方向に右左折に回転
            反対のセンサーの距離が伸びたら後退
                左右どちらかのセンサーが順番に距離が伸びる
                距離が伸びた時間を計算して自分が階段に垂直かを見る
        垂直になったら降る
    階段を降りてる最中は前進のみ

DAMBAの問題点
    解決したい
    1.下階段の判定が無理（解決できそう）
        原因
            右左折の速度が違う
        解決策
            速度に依存しない方法で下階段を判定する
        1.1左に傾く
            右の下がなくなったら止まって、
            回る、
            左も下がなくなったら止まる、
            後退(A)、
            右と左の下が床になったら止まる、
            右の下がなくなるまで前進、
            回る、
            左の下がなくなったら止まる、
            (A)に戻る、
            右と左の床になる時間の差が短くなったら垂直、
            前進
        1.2右に傾く
            1.1のそれぞれ逆
        1.3垂直で階段に入った時
            距離更新時、両方とも下にない状態になったら前進

    2.階段を登れない
        原因
            トルク？そもそもある程度腕を曲げて登るのが無理？
        解決策
            電池増やす？←通常の前進で車両が外れるかも、重くなって登れないかも
            腕を上げながら前進？←上げた後下げれないかも

    3.前進ができる時できない時がある
        原因
            モーターの空回り？どこか引っかかってる？
            モーターが回ってる音はする
        解決策
            ギアボックスの改良

    4.PC無しで起動できない（解決できそう）
        原因
            起動時にシステムが動かせない
            UIがまだない
        解決策
            起動時のシステムは外部に聞く
            UIは余ったセンサーでどうにかする
            ラズパイのライトをいじって状態の確認？できそうだけどできるかわからない

    5.右左折の回転速度が違う
        原因
            なんか引っかかってる？車体を浮かすと早い
        解決策
            左右で回る時間を変える

    6.右左折するとゴムがたくさん取れる
        解決策
            ゴムを何かで止める

    7.上階段の時、垂直にする段階で右左折で壁に当たるかも（解決できそう）
        解決策
            ある一定の距離より近くなれば交代する



解決しない問題点（パワポ用）
    1.端っこまで掃除できない
        予算と時間がない
        掃除方法を変更することや機体の形状を帰ることにより対応可能
    2.上階段のセンサーが折れそう
        予算がない
        壁か上階段を判断する時にセンサーを伸び縮みさせる
    3.外観
        予算がない
        フレームとかを3Dプリンターで作れるといい
    4.ジャンパー線とかフレットボード系がぐちゃぐちゃ
        予算と時間がない
        それ用の基盤を作れたらかなり綺麗になる
    5.中途半端な隙間が有れば、上階段と壁の判断ができない
        上階段用のセンサーを増やしてさまざまな段差に対応
